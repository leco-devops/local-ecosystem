# LEco DevOps Open Project v0.1.0

**Release date:** 2026-05-01  
**Platform version:** 0.1.0

## Summary

First documented platform release: local `*.lh` edge with Traefik, ecosystem stack services, **LEco DevOps** dashboard, **`leco-devops` CLI**, hosted apps, and optional **Cloudflare-local** adapters.

## Highlights

### Added

- Ecosystem stack orchestration (`ecosystem-stack/`), foundation installer.
- LEco DevOps dashboard: Control, Infrastructure, Metrics, Hosted apps, Routes, Logs, Docs.
- `leco-devops` CLI, `hosting/app-available/`, registry, Traefik merge.
- Cloudflare-local compose; AI onboarding; local edge runtimes; hosted-app Traefik runbook.
