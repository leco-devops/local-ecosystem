# LEco DevOps Open Project v0.3.0

**Release date:** 2026-05-16  
**Platform version:** 0.3.0  
**Git tag:** `v0.3.0` (recommended)

## Summary

Dashboard-focused release: **Updates & catalogs** on Overview is fully interactive (schedule, read state), toolbar refresh is persistent, and server-backed actions show **inline button preloaders**. Introduces the project **versioning system** and technical release documentation.

## Highlights

### Added

- Editable **update-catalog schedule** (interval hours or fixed UTC times); saves `ecosystem-stack/config/update-catalog-schedule.json`.
- **Mark all read** for stack/Ollama alerts with unread row highlighting.
- **Inline control token** field on Overview when `DASHBOARD_CONTROL_TOKEN` is set.
- **Persistent auto-refresh** interval (5s / 10s / 30s / 60s / Manual) in toolbar.
- **Inline action preloaders** on controls that trigger `/api/*` requests.
- Versioning: `VERSION`, `version.json`, `CHANGELOG.md`, `GET /api/version`, `releases/`, `docs/VERSIONING.md`, `docs/RELEASE_NOTES.md`.

### Changed

- `leco-update-catalog` watcher sleep aligns with schedule config file.

### Breaking changes

- None.

3. Hard-refresh browser (`dashboard.js` / `dashboard.css` cache query).
