# LEco DevOps Open Project v0.2.0

**Release date:** 2026-05-16  
**Platform version:** 0.2.0

## Summary

Adds **AirLLM** for large HuggingFace models, a background **update-catalog** service for stack and LLM release tracking, and a full **Help & User Manual** in the dashboard.

## Highlights

### Added

- **AirLLM** container, Traefik route `airllm.lh`, Infrastructure model manager, `leco-cli.sh airllm` commands.
- **`leco-update-catalog`** watcher: Docker Hub, Ollama library, HuggingFace trending; JSON + Help markdown tables.
- **`/help`** manual: installation, hosting, developer guide, architecture diagrams.
- Overview **Updates & catalogs** panel (read-only status); APIs `/api/ecosystem/updates`, `/api/llm-catalog/*`.

### Changed

- Infrastructure: dual LLM managers (Ollama + AirLLM); expanded docs and AGENTS guardrails.
