feat: dashboard UX — slug-to-URL auto-sync, fresh probe override, hosted app fixes

Register wizard: changing the App id (slug) now auto-updates the hostname
in all Public URL rows. Instead of tracking a previous-slug variable (which
got out of sync when Detect populated URLs from the folder name rather than
the slug), the listener reads the actual hostname from the first URL row
and rewrites all rows on each keystroke. Custom paths like /api or /graphql
are preserved.

Hosted app detail panel: the main-URL status dot now uses the fresh
url_probes from the per-app snapshot instead of the stale main_url_probe
cached in the sidebar list (90s client cache + localStorage). This fixes
the 'main URL 502' badge lingering after the app recovers.

Also includes: staging badge support, manifest summary panel, config
validation endpoint, Traefik dynamic sanitization, localhost URL extraction,
hosting overlay samples, .gitignore for per-machine app-available dirs,
documentation updates (HOSTED_APPS_TRAEFIK_RUNBOOK, DEVELOPMENT_PLAYBOOK,
LECO_USER_MANUAL, DEPLOY_CLI, and others).
